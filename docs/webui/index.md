# Web UI

Web rendering is planned for the xnano 1.0 release series. The same declarative
`Grid` and component definitions used for terminal applications will be able to
target a browser-based renderer, with request and endpoint declarations for
interactive applications.

The public API and installation guide will be documented here when the web UI
preview is ready. Until then, build against the terminal APIs described in
[Concepts](../concepts/getting-started.md) and [Components](../components/index.md).
