# xnano-core.rust.engine

``xnano_core.rust.engine`` is the core engine that powers the main ``xnano`` rendering loop and all associated terminal related
functionality. It is built natively through rust and built to interoperate with the direct ``ratatui`` and ``tachyonfx`` bindings
exported in [xnano_core.rust.native]{data-preview}.
