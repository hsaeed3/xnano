# xnano-core.rust.native

``xnano_core.rust.native`` provides the raw, standardized bindings directly mapped from the ``ratatui`` and ``tachyonfx``
rust crates, these bindings are used both internally by [xnano_core.rust.engine]{data-preview} and directly within the
``xnano`` package core.
